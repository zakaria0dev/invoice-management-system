# FAQ - Invoice Manager

## General Questions

### Q: What is Invoice Manager?
**A:** Invoice Manager is a modern, web-based invoice management system designed to help businesses create, manage, and track invoices and payments efficiently. It includes features for quotes, refunds, inventory management, and comprehensive reporting.

### Q: Is it free to use?
**A:** Try the full functionality at [http://app.ifry.ma/](http://app.ifry.ma/). For pricing and licensing details, contact the support team.

### Q: Who can use Invoice Manager?
**A:** Anyone managing invoices can benefit:
- Freelancers and consultants
- Small to medium businesses
- E-commerce stores
- Professional service firms
- Retail and distribution companies

### Q: Is my data secure?
**A:** Yes! Invoice Manager uses enterprise-grade security:
- JWT authentication with secure tokens
- bcrypt password hashing
- HTTPS/TLS encryption
- Rate limiting and DDoS protection
- Regular security audits
- Compliance-ready audit logging

## Features & Functionality

### Q: Can I create multiple invoices?
**A:** Absolutely! Create unlimited invoices with auto-incrementing numbers, different statuses, and full customization options.

### Q: Does it support quotes?
**A:** Yes! Create professional quotes and easily convert them to invoices when the client approves.

### Q: How does payment tracking work?
**A:** Record payments as they come in, add notes, and track partial payments. Get automatic overdue alerts and reminders.

### Q: Can I issue refunds?
**A:** Yes! Issue full or partial refunds, generate credit notes, and automatically apply credits to future invoices.

### Q: Does it include inventory management?
**A:** Yes! Track products, manage stock levels, monitor stock movements, and adjust inventory as needed.

### Q: Can multiple users access the system?
**A:** Yes! Support for multiple users with role-based access control (Admin, Manager, Staff).

### Q: What export formats are supported?
**A:** Export to PDF (invoices), Excel (.xlsx), and CSV formats for data analysis and accounting software.

## Technical Questions

### Q: What are the system requirements?
**A:** 
- Any modern browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- No installation required - fully cloud-based

### Q: Which browsers are supported?
**A:** All modern browsers:
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
- Mobile browsers

### Q: How is the database protected?
**A:** PostgreSQL with:
- Regular backups
- Encryption at rest
- Secure access credentials
- SQL injection prevention via ORM

### Q: Can I access it offline?
**A:** The main application requires internet connection. Local caching is supported for offline data viewing in certain scenarios.

### Q: How fast is the application?
**A:**
- Page load: < 2 seconds
- API response: < 200ms
- PDF generation: < 5 seconds
- Extremely responsive UI
